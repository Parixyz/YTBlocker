# 📚 YouTube Shorts Blocker

**Stop getting distracted by YouTube Shorts!**

This Chrome extension blocks YouTube Shorts and shows you inspiring productivity quotes instead.

## 🚀 Quick Start

👉 **[Visit the website](https://Parixyz.github.io/YTBlocker)** ← Click here for easy installation!

Or [download directly](https://github.com/Parixyz/YTBlocker/releases/download/v1.0.0/youtube-shorts-blocker.zip)

## ✨ Features

✅ Blocks YouTube Shorts automatically  
✅ Redirects Shorts visits to a dedicated "Stay Concentrated" quote page  
✅ Enable/disable blocker from extension popup  
✅ Beautiful, motivational UI  
✅ No data collection or tracking  
✅ 100% free  
✅ Super fast  

## 📖 How to Install

### Option 1: Easy Install (Recommended)

1. [Download the extension](https://github.com/Parixyz/YTBlocker/releases/download/v1.0.0/youtube-shorts-blocker.zip)
2. Unzip the file (right-click → Extract All)
3. Open Chrome and go to `chrome://extensions/`
4. Turn ON "Developer mode" (top right toggle)
5. Click "Load unpacked"
6. Select the extracted folder
7. ✅ Done!

### Option 2: From Source

1. Clone this repository
2. Open Chrome → `chrome://extensions/`
3. Enable Developer mode
4. Click "Load unpacked"
5. Select this folder

## 💡 How It Works

1. Install the extension
2. Use YouTube normally
3. Try to watch a YouTube Short
4. Get redirected to a "Stay Concentrated" quote page with an inspiring quote instead! 💪
5. Click "Back to YouTube" to continue or "New Quote" for another quote
6. Use the popup toggle any time to enable/disable blocking

## 🔒 Privacy

This extension:
- ❌ Does NOT collect any data
- ❌ Does NOT track you
- ❌ Does NOT send data anywhere
- ✅ Works 100% offline on your computer
